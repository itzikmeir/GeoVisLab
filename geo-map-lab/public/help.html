📚 מדריך למשתמש GeoVisLab

ברוכים הבאים למערכת GeoVisLab. מערכת זו משמשת לתכנון, הרצה וניתוח של תרחישים גיאוגרפיים וויזואליים.

🛠️ תכנון תרחישים (Planner)

המסך הראשי של המערכת. כאן ניתן:

  • ליצור ישויות (רכבים, הולכי רגל) על גבי המפה.
  • להגדיר מסלולים ונקודות ציון (Waypoints).
  • לשלוט בפרמטרים כמו מהירות, צבע וזמן הופעה.
  • שמירה: יצוא התרחיש לקובץ JSON לשימוש עתידי.

🏭 מחולל המוני (Batch Generator)

כלי מתקדם להמרת נתונים:

  • טעינת קובץ תרחיש (JSON) שיצרתם בתכנון.
  • המרה אוטומטית לקובץ HTML עצמאי שניתן לשלוח למשתתפים בניסוי.
  • הקובץ המיוצר מכיל את כל הלוגיקה הנדרשת להרצת הניסוי בדפדפן.

📂 ניהול קבצים (File Manager)

אזור מרכזי לניהול התוצרים שלכם. ניתן לצפות בקבצים שיוצאו, להוריד אותם שוב או למחוק גרסאות ישנות.

טיפ: בכל שלב ניתן לחזור לתפריט הראשי ע"י לחיצה על כפתור ההמבורגר (☰) בפינה הימנית העליונה.

📚 GeoVisLab User Guide

Welcome to GeoVisLab. This system is used for planning, running, and analyzing geographic and visual scenarios.

🛠️ Scenario Planning (Planner)

The main screen of the system. Here you can:

  • Create entities (vehicles, pedestrians) on the map.
  • Define routes and waypoints.
  • Control parameters such as speed, color, and appearance timing.
  • Save: Export the scenario to a JSON file for future use.

🏭 Batch Generator

An advanced data conversion tool:

  • Load a scenario file (JSON) created in the Planner.
  • Automatically convert it to a standalone HTML file that can be sent to experiment participants.
  • The generated file contains all logic needed to run the experiment in a browser.

📂 File Manager

A central area for managing your outputs. You can view exported files, re-download them, or delete old versions.

Tip: At any stage you can return to the main menu by clicking the hamburger button (☰) in the top corner.
